<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CPU.xlsx 文件具有 16 位 CPU、16 个通用寄存器、128KB RAM 和 128x128 显示屏。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">必须打开迭代计算。</font><font style="vertical-align: inherit;">这可以通过转到文件 -&gt; 选项 -&gt; 公式 -&gt; 然后启用迭代计算并将</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">最大迭代次数设置为 1 来完成</font></font></strong></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CPU 根据 B2 中设置的时钟信号运行。</font><font style="vertical-align: inherit;">该时钟信号将在 Excel 电子表格内重新计算的正常条件下更新。</font><font style="vertical-align: inherit;">按 F9 键将重新计算电子表格。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">F2 单元中的重置按钮如果设置为 true，会将 PC 寄存器重置回 0。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CPU.xlsx 文件中的计算机可以以自动或手动模式进行控制。</font><font style="vertical-align: inherit;">这是由 J2 中的按钮控制的。</font><font style="vertical-align: inherit;">如果设置为 true，则当来自 B2 的时钟信号为高电平时，CPU 将执行单元格 D8 中的取指单元中的覆盖槽中指定的操作。</font><font style="vertical-align: inherit;">如果为 false，则 CPU 将执行从 PC 寄存器指定的内存表中检索的操作。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Reset RAM 按钮，如果设置为 true，会将每个内存单元重置为 0。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果将“读取 ROM”按钮设置为 true，则会将 ROM.xlsx 电子表格中的内存表的值复制到 CPU.xlsx 电子表格的 RAM 表中。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CPU 的正常操作包括将 Reset 按钮设置为高电平，再次打开和关闭 Reset RAM 或 Read ROM 按钮（导致 RAM 重置或 ROM 读入 RAM 表），然后关闭重置按钮。</font><font style="vertical-align: inherit;">然后，CPU 将被设置为在手动模式下运行程序，或者执行 RAM 中指定的程序。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CPU 设计为根据InstructionSet.xlsx 电子表格中指定的指令集架构运行。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">警告：简单地按 F9 键是不可能尽快完成的，Excel 更新这么多单元格需要时间，建议等到 Excel 左下角看到“Ready”文本后即可继续按 F9 键之前可以看到。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">或者，可以使用 Excel-ASM16 语言编写程序并将其编译为 ROM.xlsx 电子表格。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Excel-ASM16 具有 24 种不同的不区分大小写的指令。</font><font style="vertical-align: inherit;">每条指令使用三个不同的操作数</font></font></p>
